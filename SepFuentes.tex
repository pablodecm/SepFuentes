    \subsection{AnÃ¡lisis de Componentes Principales}


    El AnÃ¡lisis de Componentes Principales (PCA, del inglÃ©s), es un

SupÃ³ngase que se tiene una matrÃ­z de datos \(\mathbf{Y}\) (con

\[ \mathbf{C_Z} = \mathbf{Z} \mathbf{Z^T} = \mathbf{P} \mathbf{Y} (\mathbf{P} \mathbf{Y})^T =

Dado que la matrÃ­z \(\mathbf{P}\) es ortogonal, su tranversa es igual a

mayor y asÃ­ sucesivamente.

Este mÃ©todo es Ãºtil para la compresiÃ³n y representaciÃ³n de datos, ya que
permite cambiar a una base en la que las componentes estÃ©n
